# GUI Calculator

<strong>This is a simple graphical user interface builded with python, using the tkinter library where we can make math operations<strong>

<br/>

## Requirements

```
pip install -r requirements.txt
```

## Usage

```
make start
```

or if Makefile is not enable

```
python3 main.py
```
